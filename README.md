# Advent of Code 2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Python.

## Setup

```bash
# Install dependencies
pip install pytest

# Run tests for a specific day
pytest tests/test_day01.py

# Run all tests
pytest tests/
```

## Running Solutions

Each solution is a Python file with `#%%` cell markers for interactive execution in editors like Zed or VS Code.

```bash
# Run a complete day's solution
python solutions/day01.py
```

Or open in Zed and run cells interactively with `ctrl-shift-enter`.

## Structure

- `solutions/` - Daily puzzle solutions
- `tests/` - Pytest tests for each day
- `utils/` - Shared utility functions
- `resources/` - Puzzle descriptions and inputs (not in git)
